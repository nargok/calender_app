# 動作確認手順

## ソースコードをクローンする
```
$ git clone https://github.com/nargok/calender_app.git
$ cd calender_app
```
## アプリケーションを立ち上げる
```
$ yarn install
$ yarn start
```
## ローカルでの動作確認
ブラウザで、[http://localhost:3000](http://localhost:3000)にアクセスしてください。
